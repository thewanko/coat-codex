# coat-codex — 画面テンプレート

> 本書はcoat-codexのビジュアルデザイン検討（Claude Design）用資料の一部です。3点セット: [依頼プロンプト](coat-codex_ClaudeDesign_プロンプト.md)／[DesignSystem初期値](coat-codex_DesignSystem.md)／[画面テンプレート](coat-codex_画面テンプレート.md)。機能仕様の原典は [技術計画v2](../coat-codex_技術計画_v2.md)。



> 各画面の構造ラフ。**配置順と要素の有無は機能設計確定分**、造形・密度・装飾はClaude Design裁量。凡例: `▤`=ドラッグハンドル ／ `■`=SwatchChip ／ `●`=未バックアップドット ／ `⋮`=カードメニュー。共通ブレークポイント768px。

| 状態 | 表示 |
|---|---|
| 新規（空） | 全欄空。make codex!は有効（タイトル空なら既定名を補完する前提でボタンは無効化しない） |
| 入力済み | autosave（500ms debounce）。保存ボタンなし |

## C-8. 全画面共通の状態規約

- **autosave**: 全編集画面で500ms debounce保存。保存ボタン・未保存警告は存在しない。失敗時のみerrorトースト
- **ロード**: IndexedDB読込中はSkeleton（スピナー単独より紙面骨格を優先）
- **不正ルート**: 存在しない `:id`/`:partId` はHome/Overviewへリダイレクト＋トースト
- **写真**: すべて非加工表示（フィルタ・トーン変更禁止）、角丸 `--radius-lg` のみ許可
